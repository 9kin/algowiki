Дано число N и последовательность из N целых чисел. Найти вторую порядковую статистику (второй минимум) на заданных диапазонах.

Формат файла входных данных:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

В первой строке заданы 2 числа: размер последовательности N  (1 ≤ N ≤ 105) и количество диапазонов M  (1 ≤ M ≤ 4*106).

Следующие N целых чисел задают последовательность. Далее вводятся M пар чисел - границ диапазонов.

Гарантируется, что каждый диапазон содержит как минимум 2 элемента.

Все числа не превышают 109 по модулю.


Формат файла выходных данных:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Для каждого из M диапазонов напечатать элемент последовательности - 2ю порядковую статистику. По одному числу в строке.
